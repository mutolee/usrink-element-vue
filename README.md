### 如何安装

下载项目之后，在文件夹 vel-admin-web 下执行以下命令，进行安装依赖包。

```
npm install
```

### 如何运行

安装依赖包的时候，会在 vel-admin-web 文件夹下生成 node_modules 文件夹，里面放的就是被安装的依赖。

依赖都安装完成后，执行以下命令，运行本项目。

```
npm run serve
```

### 如何打包

项目完成之后，要编译打包，执行以下命令。

```
npm run build
```

命令执行完会在 vel-admin-web 文件夹下生成 dist 文件夹，里面就是css、img、js、html等等文件，后面就不用说了吧！！！
